python_mcc_api
==============

A wrapper for the `MC Championship API <https://api.mcchampionship.com>`_, inspired by `derNiklaas's <https://github.com/derNiklaas>`_ `node-mcc-api <https://github.com/derNiklaas/node-mcc-api>`_ project.

- Issues: `https://github.com/JamesMCo/python_mcc_api/issues <https://github.com/JamesMCo/python_mcc_api/issues>`_
- MCC API documentation: `https://api.mcchampionship.com/docs <https://api.mcchampionship.com/docs>`_
- Module documentation: `https://mrjamesco.uk/python_mcc_api <https://mrjamesco.uk/python_mcc_api>`_
- PyPI: `https://pypi.org/project/mcc-api/ <https://pypi.org/project/mcc-api/>`_
- Repository: `https://github.com/JamesMCo/python_mcc_api <https://github.com/JamesMCo/python_mcc_api>`_
